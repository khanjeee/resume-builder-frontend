# Resume Builder Frontend

This repository contains the frontend application for the Resume Builder.

## Setup Instructions

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/khanjeee/resume-builder-frontend.git
    cd resume-builder-frontend
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Environment Variables:**
    Create a `.env` file in the root directory based on `.env.example` and fill in the necessary values.
    ```
    PORT=3000
    REACT_APP_API_URL=http://localhost:5001 # Replace with your backend API URL
    ```
4.  **Run the application:**
    ```bash
    npm start
    ```

The frontend application will be running on `http://localhost:3000` (or your specified PORT).
